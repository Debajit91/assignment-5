Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM. These are: getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectroAll, getElementsByName etc. The most using methods are: getElementById, querySelector, querySelectroAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

By using innerHTML we can access or set the HTML tag inside an element.
By using innerText we can access or set the text of an element and
using textContent we can access or set all the text content including HTML tag of an element.

Question-3: What is event delegation in the DOM?

The event delegation is a powerful technique by which we can control all the child element of a parent element by adding only one eventListener. In this way we have not to add seperate eventListener for all the child element, only adding single eventListener to the parent element.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a mechanism of DOM where the events that occur on an element such as clicks, mouseovers etc. flow upward from that element to its parent element.

Question-5: How do you create, add, and remove elements using JavaScript?

To create elements I use document.createElement() method.
To add elements I use appendChild(), append() and prepend().
To remove elements I use removeChild() or remove().
